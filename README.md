
# Job Posting Website for Recruiter






- Use React Hooks
- Use Axios to Post data
- Used Bootstrap for designing



## Run Locally

Clone the project

```bash
  gh repo clone subh-23/Extract-habitable-planets-Js
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
